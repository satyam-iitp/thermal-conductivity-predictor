# 🧪 κ_L Predictor — Lattice Thermal Conductivity Web App

Predicts lattice thermal conductivity (W/mK) for crystalline compounds using a
LightGBM model trained on 5,530 DFT-computed materials from the AFLOW database.

---

## 📁 File Structure

```
thermal_conductivity_app/
├── app.py                          ← Flask web app
├── train_model.py                  ← Train & save the model (run once)
├── requirements.txt                ← Python dependencies
├── thermal_conductivity_dataset.csv← Training data
├── kl_predictor_lgbm.pkl           ← Saved model (generated by train_model.py)
├── kl_top10_features.pkl           ← Top-10 feature names
└── feature_stats.pkl               ← Dataset stats for UI hints
```

---

## ⚙️ Setup & Run (Local)

### 1. Install dependencies
```bash
pip install -r requirements.txt
```

### 2. Train the model (first time only)
```bash
python train_model.py
```
This reads `thermal_conductivity_dataset.csv`, trains LightGBM, and saves:
- `kl_predictor_lgbm.pkl`
- `kl_top10_features.pkl`
- `feature_stats.pkl`

> If you already have your `.joblib` / `.pkl` model files, skip this step and
> rename them to match the names above (or update paths in `app.py`).

### 3. Start the web app
```bash
python app.py
```
Open **http://localhost:5000** in your browser.

## 🚀 Deploy to Cloud

### Option A — Render (free tier)
1. Push this folder to a GitHub repo
2. Go to [render.com](https://render.com) → New → Web Service
3. Set **Build Command**: `pip install -r requirements.txt && python train_model.py`
4. Set **Start Command**: `gunicorn app:app`
5. Add `gunicorn` to `requirements.txt`

### Option B — Railway
1. Push to GitHub
2. [railway.app](https://railway.app) → New Project → Deploy from GitHub
3. Set start command: `python train_model.py && gunicorn app:app`

### Option C — Hugging Face Spaces
1. Create a Space with **Gradio** or **Docker** SDK
2. Upload all files; Spaces will auto-install from `requirements.txt`

### Option D — Docker
```dockerfile
FROM python:3.11-slim
WORKDIR /app
COPY . .
RUN pip install -r requirements.txt
RUN python train_model.py
EXPOSE 5000
CMD ["python", "app.py"]
```
```bash
docker build -t kl-predictor .
docker run -p 5000:5000 kl-predictor
```

## 🔬 Model Details

| Property | Value |
|---|---|
| Algorithm | LightGBM (gradient boosting) |
| Features | Top-10 by SHAP importance |
| Target | log(1 + κ_L) → expm1 → W/mK |
| Training data | 5,530 AFLOW compounds |
| Split | 80/10/10 train/val/test |

### Top-10 Features Used
Selected by SHAP importance from the full feature set:
- `Egap_fit` — DFT band gap (eV)
- `energy_atom` — DFT total energy per atom (eV/atom)
- `enthalpy_atom_x` — Formation enthalpy per atom
- `volume_atom` — Unit cell volume per atom (Å³)
- `density_x` — Mass density (g/cm³)
- `natoms_x` — Atoms per unit cell
- `nspecies_x` — Number of chemical species
- `valence_cell_iupac` / `valence_cell_std` — Valence electron count
- `eentropy_atom` — Electronic entropy per atom
- *(actual top-10 may vary slightly based on your training run)*
